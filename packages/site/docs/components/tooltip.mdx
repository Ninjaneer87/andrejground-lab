# Tooltip

A Tooltip is a small, non-interactive overlay that displays brief informational text when hovering over or focusing on an element. It is built on top of the <Link to='/docs/components/popover'>Popover</Link> component.

## Key Characteristics

- **Hover-Triggered**: Automatically shows on hover with configurable delay.
- **Non-Interactive**: Unlike Popover, tooltips are meant for displaying simple text hints, not interactive content.
- **Accessible**: Works with keyboard focus and follows accessibility best practices.

## Import

```tsx
import { Tooltip } from '@andrejground/lab';
```
